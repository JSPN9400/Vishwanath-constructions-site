1) Clear Quotation (Scope + GST)

  • Work scope clearly written ho: structure / finishing / all-work.
  • Material mode clear ho: with material ya without material.
  • GST breakup aur billing transparency (GSTIN) check karein.

2) Site Supervision + Updates

  • Daily/weekly site updates (photos/videos) ka process ho.
  • Stage-wise checks: RCC, brickwork, plaster, waterproofing, finishing.

3) Material Quality + Brand Discussion

  • Cement/steel/brick/sand/tiles/paint—brands and specs pe discussion.
  • Site delivery verification aur quantity tracking.

4) Timeline + Payment Milestones

  • Planned timeline with milestones (foundation → structure → finishing → handover).
  • Milestone-based payments—clear documentation.
